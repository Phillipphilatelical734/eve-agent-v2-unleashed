## What is Eve Agent V2 Unleashed?

Eve Agent V2 Unleashed is a local-first autonomous coding assistant. It runs directly on your computer. You do not need to send your code to the cloud or rely on external servers. The agent uses local intelligence to help you write, edit, and fix code. It acts as an autonomous assistant that manages your programming projects.

## 🧠 Setting Up the Brain

Eve Agent uses Ollama to power its local intelligence. You must install Ollama before you use the agent.

1. Go to the Ollama website and download the Windows version.
2. Run the installer.
3. Open your command prompt or search for Ollama in your start menu.
4. Type `ollama run llama3` to download the intelligence model.
5. Wait for the progress bar to finish.

## 🚀 Running Your Agent

Once you finish the installations, you can start the application.

1. Open the Eve Agent application from your desktop or start menu.
2. The software will detect the Ollama service running on your machine.
3. A command window will appear. It shows the status of your agent.
4. Open your web browser if the interface does not appear automatically.
5. Type `http://localhost:8000` into your address bar to access the control panel.

## 🛠️ Using the Coding Assistant

The dashboard gives you full control over your projects. You can point the agent at a folder on your computer that contains your code files.

- Select a project folder.

- Ask a question or give an instruction in the text box.
- The agent reads your files and suggests changes or writes new functions.
- Review the suggested code.
- Click apply to save the changes to your project.

## 🛡️ Privacy and Safety

This software keeps your data private. Since the agent runs locally, your code never leaves your computer. The model does not phish for information or send your work to a remote database. Every process occurs within your own file system and memory.

## 🔧 Troubleshooting and Support

If the agent does not respond, verify your settings.

- Check if Ollama works by typing `ollama list` in your command prompt.
- Ensure your antivirus software does not block the application.
- Redownload the latest version if errors persist.
- Make sure your GPU drivers are up to date.

- `bin/`: Contains the main application files.

- `logs/`: Keeps a record of recent agent actions.
- `config/`: Contains your user preferences and settings.

## 💡 Best Practices

You receive the best results when you break down complex coding tasks. Ask the agent to write one small function at a time rather than an entire application. Review the code after each step. You maintain control of the project logic while the agent handles repetitive or time-consuming tasks.
